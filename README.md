# Electronics Component Organizer Labels

Based on the [Ziplock Bag-based Component Organizer](https://www.youtube.com/watch?v=N9kQCDN8lkk) ([github](https://github.com/Cristian-Ariel/Ziplock-Bag-Based-Component-Organizer/))


Implemented in [Typst](https://typst.app/) to be more flexible and easier to add to.

# Building Individual Label Files

* Install Typst `brew install typst`
* Generate a PDF `typst compile src/resistors.typ`
